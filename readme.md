# 异星工场产量计算工具

## 快速开始

获取：

```shell
git clone git@github.com:jcq15/factorio.git
```

安装依赖：

```shell
pip install -r requirements.txt
```

运行后端：

```shell
python backend.py
```

打开 `127.0.0.1:5000/` 即可。
